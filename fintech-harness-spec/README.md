# 핀테크 하네스 엔지니어링 스펙

이 폴더는 `ai-repo`의 초기 합의 내용을 스펙 문서로 고정하기 위한 작업 결과물입니다.

## 문서 목록

- `fintech-harness-engineering-spec.md`: 운영 하네스와 포트폴리오형 핀테크 학습 레포의 기획 스펙

## 현재 합의 요약

- 목표는 `운영 하네스`와 `포트폴리오형 결과물`을 함께 달성하는 것입니다.
- 문서 품질이 가장 중요하며, 사용자는 문서를 기준으로 구현과 운영을 검증합니다.
- 첫 기능 흐름은 `잔액/거래내역 → 회원/지갑 계정 → 충전/송금` 순서입니다.
- 중요한 결정의 기준 문서는 `ADR`입니다.
- 전체 사고 과정과 역할별 논의 로그는 `GitHub Wiki` 중심으로 남깁니다.
- 초기에는 하네스 역할을 촘촘하게 적용하고, 반복 패턴이 안정되면 체크리스트 중심으로 경량화합니다.
